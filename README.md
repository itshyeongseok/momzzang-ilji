# 몸짱일지 💪

운동·식단·생활습관·신체변화를 한곳에 기록하는 **개인 헬스 일지** (PWA).

- 로그인·서버·결제 없음. 데이터는 내 폰(브라우저)에만 저장됩니다.
- 오프라인에서도 동작하고, 홈 화면에 추가하면 앱처럼 실행됩니다.

## 사용

- 바로 쓰기: GitHub Pages 주소를 폰 브라우저에서 열고 **"홈 화면에 추가"**
- 로컬에서 테스트: `python serve.py` 후 `http://localhost:5777/index.html`

## 백업

⚙(설정) → **백업 파일 내보내기** 로 JSON 저장. 폰을 바꾸면 같은 화면에서 불러오기.
