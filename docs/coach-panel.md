# BULCUP 헬스 코치 패널 (= AI 코치 = 헬스 관리팀)

고형석의 몸·운동·식단 데이터를 받아 **세계적 코치·의사·영양 연구원이 종합 진단·제안**하는 팀. 형석님의 Claude **구독**으로 동작(ANTHROPIC API 키 0원).

## 구성
| 역할 | 정의 | 관점 |
|---|---|---|
| **헤드 코치** | `.claude/agents/coach-head.md` | 패널 종합 → 실행 가능한 주간 리포트 |
| 스트렝스 코치 | `.claude/agents/coach-strength.md` | 점진적 과부하·부위균형·빈도/회복 |
| 스포츠의학 의사 | `.claude/agents/coach-doctor.md` | 회복·피로·부상위험(진단 아님, 위험 시 전문의 권고) |
| 영양 연구원 | `.claude/agents/coach-nutrition.md` | 단백질·칼로리·근성장 식단 |

## 운영 흐름 (주간 리포트)
```
고형석 기간 데이터(운동/식단/습관/신체)
  → 스트렝스 ∥ 의사 ∥ 영양  (병렬 분석)
  → 헤드 코치가 종합 → 한국어 코칭 리포트
  → (자동화 시) 앱 '분석' 탭에 표시
```

## 데이터 통로 — 현재 vs 자동화
- **현재(Supabase 보류 중)**: 형석님이 앱에서 **백업 내보내기(JSON)** → 그 데이터로 패널을 돌려 리포트 제공. (운동/식단이 ~1주 이상 쌓이면 의미 있는 리포트가 됨.)
- **자동화(Supabase 재개 후)**: 매주 **일요일 밤** Claude Code 예약 루틴이 Supabase에서 고형석 7일 데이터를 읽어 패널 가동 → 리포트를 저장 → 앱 분석 탭에 자동 표시. (service_role 키는 그 루틴의 로컬 PC에만, 공개 금지.)

## 안전 원칙
- 의학적 **진단 단정 금지** — 웰니스 가이드 + 위험 신호 시 실제 전문의 상담 권고.
- 데이터 근거. 부족하면 솔직히. 입문자(고형석)에 맞춰 쉽게. 과장·만능 주장 금지.
